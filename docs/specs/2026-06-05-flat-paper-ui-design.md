# Atoman 扁平纸叠空间设计系统规范 (Flat Paper-Stacking UI Design Spec)

本规范旨在将 Atoman 网站的 UI 风格重构为扁平、干净且极具物理纸质隐喻的系统。我们移除所有冗余的阴影和厚重的黑边，保留文字阅读的高可读性，通过极细的浅灰边界线与极其克制的淡雅色彩来构建页面。

---

## 1. 核心视觉语言 (Core Visual Language)

### 1.1 纯白底色与极细边界
* **纸张基板 (Paper Base)**：全站背景和所有的内容卡片、弹窗、Sheet 均保持纯白 (`#FFFFFF`)，保证文字在各类屏幕下的极致易读性。
* **极细浅灰边界 (Thin Borders)**：移除了此前 2px 的粗黑边框，全站统一使用 **1px 浅灰色/板岩灰实线** (`#cbd5e1` 或 `var(--a-color-line-soft)`) 进行视觉分割。
* **无阴影设计 (Zero Shadows)**：移除了所有弥散阴影和硬色块投影。通过边界线、间距 (Padding/Margin) 和对齐关系来表达纸张的逻辑层次。
* **无装饰性图标 (No Decorative Icons)**：禁绝添加任何非功能性/装饰性的 SVG 图标和 Emoji。侧栏导航、列表、页签等一律采用纯文本或数字排版，展现严谨的印刷体感。

### 1.2 低调的莫兰迪彩色点缀 (Quiet Accents)
在纯白与浅灰的基础上，使用莫兰迪/粉色系进行低调的点缀：
* **主要动作/高亮态 (Primary Active)**：激活的导航链接、选中的侧栏菜单，使用极淡的淡蓝色/淡灰色背景 (`#f1f5f9` 或 `#bfdbfe` 等)，并搭配 1px 稍深色边框。
* **分类标签 (Badges)**：博客、播客、视频等分类标签，采用无背景色或极淡底色（如淡绿、淡紫、淡红），配以细灰色边框与小字重暗色字。
* **马克笔高亮标记 (Highlighters)**：
  * **收藏**：在标题/文字下覆盖一笔极淡的马克笔黄色底色与亮黄色底线 (`#fefce8` 背景 + `#fef08a` 底边线)。
  * **稍后阅读**：在文字下挂极淡的青色底色底线 (`#f0fdfa` 背景 + `#99f6e4` 底边线)，或者灰色细点状下划线。

## 2. 导航与多层纸张堆叠交互 (Navigation & Stacking Logic)

### 2.1 保持侧边栏条目原样
* 侧边栏项目（包括其名称、路由、顺序、分组结构等）**保持全站现有代码的实现不变**，不进行翻译、改名或删改。
* 仅重构侧栏项目的渲染样式：去除圆角与背景，采用 1px 实线框，激活态使用淡淡的莫兰迪背景。

### 2.2 电脑端：无限层级叠纸 (PC Infinite Stacking)
* **滑入覆盖**：当打开详情、作者 Wiki、或者次级页面时，新的 Sheet 从屏幕右侧滑入覆盖，遮挡前一个页面。
* **左侧索引边缘 (Sheet Index Bar)**：每一个处于底层的 Sheet 都会在左侧露出一个约 `32px` 宽度的窄白条/边缘把手，标有纵向大写文本（如 `SHEET 01 / ARTICLE`）。
* **抽离与返回**：
  * 点击底层的把手，可直接将覆盖在上面的 Sheet 向右滑走，退回那一层。
  * 支持在详情页面向右拖动/滑动（Swipe Right）直接推走页面返回。

### 2.3 移动端：双页级限制 (Mobile Stacking Limit)
* 堆叠层级硬性限制为最多两层（底层：主列表页；顶层：当前详情页）。
* 在详情页内点击任何新链接，都在当前顶层原位载入刷新，不开启新的 Sheet。
* 用户通过点击左侧返回栏（`24px` 宽度）或向右滑动，即可推走当前页退回列表。

---

## 3. 操作与删除隐喻 (Actions & Deletion)

### 3.1 收藏与稍后阅读
* **高亮标色**：被标记的文章标题在 Timeline 列表中直接应用马克笔高亮，用户可以一眼识别出已收藏或待读的内容。
* **交互反馈**：点击卡片底部的 `PaperClip` (回形针) 操作项时，文字即刻被高亮覆盖。

### 3.2 移除与删除
* **直接删除机制**：移除了所有占位印章或置灰中间态。点击“删除”或“退阅”时，项目直接从列表或详情堆叠中移出（配合简单的扁平淡出动效），不再在屏幕上残留多余的作废卡片，维持界面的极度干净。

---

## 4. 实施路径 (Migration Path)

1. **全局样式更新**：
   * 修改 `web/src/style.css`，将 `--a-border-width` 调整为 `1px`，将颜色变量更新为更柔和的 Slate 灰色系。
   * 移除 `style.css` 及组件内的所有 `box-shadow`。
2. **重构顶部栏与按钮**：
   * 去除顶部栏所有圆角元素 (`border-radius: 9999px`)。
   * 重构用户按钮 `user-btn` 和频道下拉框 `channel-select` 的直角与边框样式。
3. **实现 PC 端多层滑动 Sheet**：
   * 改进 `PaperSheet.vue` 组件，使之支持接收层级索引，并在左侧渲染竖向索引把手与向右滑动手势。
4. **统一 Timeline 马克笔标记**：
   * 在 `PaperEntry.vue` 内部引入收藏与稍后阅读的高亮 CSS 类，渲染干净的水彩高亮底色。
