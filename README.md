# Atoman-Frontend

Atoman 的 Vue 3 前端应用，负责 Studio、RSS、论坛、辩题、音乐档案库等页面与交互。

## 技术栈

- Vue 3
- Vite
- TypeScript
- Pinia
- Vue Router
- Tailwind CSS
- Playwright
- Vitest

## 常用命令

```bash
bun install
bun run dev
bun run build
bun run type-check
bun run test:unit
bun run test:e2e
```

## 优先级判断原则

- **先补刚需**：优先补足搜索、过滤、自动化这类高频能力
- **再做增强**：在基础工作流稳定后，再加强阅读体验与探索能力
- **最后做扩展**：AI、多源接入、跨端开放能力放在中长期推进
- **保持差异化**：持续强化“站内内容 + 外部 RSS”统一时间线，而不是只做通用 RSS Reader
