# Atoman-Frontend

Atoman 的 Vue 3 前端应用，负责 Studio、RSS、论坛、辩题、音乐档案库等页面与交互。

## 技术栈

- Vue 3
- Vite
- TypeScript
- Pinia
- Vue Router
- Tailwind CSS
- Playwright
- Vitest

## 常用命令

```bash
bun install
bun run dev
bun run build
bun run type-check
bun run test:unit
bun run test:e2e
```

## 环境文件

前端保留三套环境文件：

- `.env.example`：模板
- `.env.dev`：本地开发
- `.env.prod`：生产构建

当前脚本已经按 mode 区分环境：

```bash
bun run dev
bun run build
```

对应关系：

- `bun run dev` 使用 `.env.dev`
- `bun run build` 使用 `.env.prod`

本地开发应使用：

```env
VITE_API_URL=http://localhost:8080/api
VITE_TURNSTILE_SITE_KEY=
```

生产环境应使用：

```env
VITE_API_URL=https://api.atoman.org/api
```

## 优先级判断原则

- **先补刚需**：优先补足搜索、过滤、自动化这类高频能力
- **再做增强**：在基础工作流稳定后，再加强阅读体验与探索能力
- **最后做扩展**：AI、多源接入、跨端开放能力放在中长期推进
- **保持差异化**：持续强化“站内内容 + 外部 RSS”统一时间线，而不是只做通用 RSS Reader
